## 内部物品

按照传统, 该文件夹 ("internal") 用于存放 Koish 内部使用的物品, 也就是那些不会出现在玩家认知里的物品.

按照传统, 该文件夹下面又进一步分为了以下几个文件夹:

- `menu`: 用于菜单中的图标
    - `common`: 可同时用于多个菜单的图标
    - `blacksmith`: 收购站与修复台
    - `crafting_station`: 合成站
    - `merging_table`: 合并台
    - `modding_table`: 定制台
    - `rerolling_table`: 重造台
    - `core_icon`: 物品核心的图标
